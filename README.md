## How to Use

1. **First, create your account** with name, email, and password
2. **Deposit money** to add funds to your account  
3. **Withdraw money** when needed (with balance validation)
4. **Check transaction history** anytime to see all activities
5. **Modify settings** to update your account information
6. **Enter 0** anytime to stop deposit/withdrawal operations
